# Agenda Abril Movil

V1 inicial de la app movil con Expo y TypeScript. Esta iteracion usa solo pantallas mock y datos locales; no consume backend ni implementa login real.

## Ejecutar

```bash
cd app-movil
npm install
npm start
```

Luego escanea el QR con Expo Go.

## Alcance actual

- Pantalla visual de ingreso mock.
- Pantalla "Mis actividades de abril" con datos locales.
- Navegacion minima por estado dentro de `App.tsx`.
- Configuracion preparada para una futura URL de backend en `src/config/environment.ts`.
